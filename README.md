# **Template react native**
## **Installations**
+ Clone project
+ ***For Dev*** Install dependencies for watch node process, run from root folder
    ```
    npm install
    npx pod-install
    cd ./ios
    pod install
    ```
### Run ios by xcode
### Run android by android studio
